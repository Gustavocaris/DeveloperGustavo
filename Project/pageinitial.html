Sobre mim:

Comecei minha jornada acadêmica aos 17 anos, cursando Análise e Desenvolvimento de Sistemas. No entanto, precisei interromper meus estudos temporariamente devido ao serviço militar (mais detalhes na linha do tempo abaixo). Anos depois, retomei os estudos iniciando do zero um novo curso de Engenharia de Software.

Meus primeiros projetos foram focados no desenvolvimento front-end de páginas web, utilizando HTML, CSS e JavaScript. Com o tempo, passei a criar interfaces com React, o que me levou a aprofundar meus conhecimentos em JS, design de experiência do usuário (UX) e acessibilidade.(UX)

Atualmente, estou avançando minhas habilidades em conceitos de arquitetura de software e engenharia, otimizando e praticando controle de versionamento com Git, estudando metodologias ágeis como Scrum e Kanban, e desenvolvendo alguns projetos de Data Science com Python, SQL, explorando Machine Learning, buscando obter conhecimentos valiosos para aplicar no mercado profissional e me desenvolver.

Skills: HTML/CSS JavaScript React Git Python SQL Power BI Scrum Kanban
